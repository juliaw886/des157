Needfinding

  1. The site needs to have a clear navigation system in place
  2. There needs to be an ease of navigation in which one set of content leads to the next
  3. There needs to be an incorporation of media that does not require the user to click on a video or hover to learn more
  4. The site needs to translate onto mobile
  5. There needs to be strong visual hierarchy
  6. There needs to be eye-catching visuals and colors
  7. The text needs to be succinct and easy-to-read
  8. The tone of the website needs to be casual, not too serious
  9. There needs to be a hook (text, video, image) on the landing page that makes people curious
  10. There needs to be something specific that the user can gain from this (knowledge, entertainment) for them to want to stay on the page longer
  11. There needs to be a good indication of what kind of content is on each page so the user knows what to except
